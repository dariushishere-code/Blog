# Portfolio My Self

A personal frontend portfolio website showcasing skills, a reading library, music player, and contact links. The design uses pixel-inspired styling and responsive layout for browser compatibility across desktop and mobile devices.

## Features

- Responsive portfolio layout with a fixed sidebar navigation
- Custom audio player with track list, album art, progress bar, and playback controls
- Light/dark theme toggle with saved preference
- Animated header text and preloader overlay
- Book library section with external links to featured titles
- Downloadable resume link and contact buttons

## Built With

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html` - main website markup
- `css/style.css` - site styling, responsive breakpoints, theme variables
- `js/script.js` - interaction logic, audio playback, theme handling, preloader
- `fonts/` - custom font files
- `image/` - album cover and image assets
- `music/` - local audio files for the music player
- `resume/` - downloadable resume PDF
- `book images/` - book cover images for the library section

## Usage

1. Open `index.html` in a browser.
2. Use the sidebar links to navigate the page.
3. Control music playback with the on-screen buttons and progress slider.
4. Toggle the theme using the button in the header.

## Browser Compatibility

The site is designed to work across modern browsers and devices. It includes CSS vendor rules for range input styling and responsive layout adjustments for smaller screens.

## Notes

- Audio playback depends on local media file availability in the `music/` folder.
- External book links open in a new tab.
- The resume download link points to the PDF file in the `resume/` folder.
